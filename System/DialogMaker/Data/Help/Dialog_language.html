Control
Description
New
 Creates new language file. After cration, the language is selected automatic in the Language Selecter combo (The combo in the right-down corner of the window)
Change
 Changes the selected word meaning. You must select a valid word from the list above to use this function
Language Selector
 Select the language which you want to edit
Tree
 When you are already selected a language choose the window which language you want to edit. If you choose "About" you can change the author and the version of the selected language from the Language Selector (The combo below)
Main List
 This is the place there you will edit the words. First part (Default) shows the meaning of the word in English. The second part (Translated) shows the translated meaning of the Default word in the first part of the list. To change the translated meaning of the word you must double-click on it or to select it and click "Change" button
