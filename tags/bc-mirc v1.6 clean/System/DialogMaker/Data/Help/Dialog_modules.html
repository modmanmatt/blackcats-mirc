Control
Description
Load
 Loads the selected module from the list above
Adjust
 Executes the default adjust command for the selected module (This command is set when the module was created)
Delete
 Deletes the selected module
Refresh
Refreshesh the modules list. Already loaded modules have different icon in front of them than the others.
Pointing at a module
Shows information about the pointed module (This information is set when the module was created)
