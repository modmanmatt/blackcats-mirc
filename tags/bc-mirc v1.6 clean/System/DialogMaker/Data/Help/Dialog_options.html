Menu

General
Item
Main
Open Last Project
 On start Dialog Maker opens the last used project
Close Loading
Closes the Loading Dialog automatically when the loading ends
Remember Positions
When you move any of the Dialog Maker's windows they save their position and when you open them again they take their last position
Show Tips
Enable or disable the Tip dialog at Dialog Maker's startup
Item

Patterns
Add
 Adds new pattern to the list
Delete
 Deletes the selected pattern from the list
Rename
 Renames the selected pattern
Refresh
 Refreshes the list with patterns
Editbox
 When you select a pattern from the list above here appears the text of the pattern (When you select a pattern from the Code Editor in the edit area appears the text that is written here)
Menu

Editors
Item
Code Editor
Red, Green, Blue
 After you select a item from the combo in the up right corner you can adjust its color by these three scollbars and editboxes. You will see the preview of the set colors in the editbox below the combo.
Font Button
When you click the button a font window appears. From it you can select the font, size and style(s) of the text in the Code Editor. You will see a preview of the made changes in the editbox above.
Item
Visual Editor
Open
 Opens Visual Editor (It usually starts when is opened or created new project)
On Top
 Makes Visual Editor On Top of all windows
Item
Control Editor
Auto Start
 Opens Control Editor when Dialog Maker starts
On Top
 Makes the Control Editor dialog On Top of all windows
Menu

Skins
Load
 Loads selected skin from the list above (You can double-click on the any skin to load it too)
Refresh
 The button in the right side. This button refreshes the list with skins
Delete
 Deletes the selected skin. The default skin cannot be deleted
Menu

Languages
Load
 Loads selected language from the list above (You can double-click on the any language to load it too)
Refresh
 The button in the righ side. This button refreshes the list with languages
Delete
 Deletes the selected language. The English language cannot be deleted because it is the default language. If you want to create your own language, please click on the Tools button again, but now choose Language Editor. From there you can edit alredy existing languages
