Control
Description
New
 Creates new Dialog Maker project
+ Open
Opens saved project (Dialog Maker files are with extension *.dm)
++ Dialog Grabber
 Open Dialog Grabber™ 5.9 by Necroman (necroman@wot.net). This program can grab dialogs from *.exe files
++ Dialog Maker
 Opens saved Dialog Maker™ project (Dialog Maker files are with extension *.dm)
++ Dialog Studio
 Opens saved Dialog Studio™ project (Dialog Studio files are with extension *.ds)
Save
 Creates or saves your dialog (The Name and X,Y,W and H positions and sizes of the dialog must be setted first)
Code Editor
 Opens Code Editor (Before that, you must create or open project)
Control Editor
 Opens Control Editor (From there, you can edit created controls in the dialog)
+ Dialog Code
 Opens current dialog mIRC code
++ View
 Opens current dialog mIRC code
++ Copy
 Copies current dialog mIRC code to the clipboard
+ Tools
 Opens Tools Menu
++ Options
 Opens Options dialog
++ Update
 Opens Dialog Maker Update dialog
++ Modules
 Opens Modules dialog
++ Language Editor
 Opens Language Editor dialog
Preview
 Opens current dialog's preview
Help
 Opens this help file
Control Buttons
 The Control Buttons are: Button, Edit, Text, Link, Box, Scroll, Tab, Icon, Check, Radio, List and Combo. With these buttons you can add controls in your dialog by clicking on one of the buttons first, and after that clicking on the Visual Editor to place the selected control

X

 If you are clicked on any control button (One of the buttons showed above - Button, List, Check...) and you don't want to add the selected control, just click on this button to unset the selected control
Status Bar
 In the first column the statusbar shows important information and in the second it shows current project and it's size
Name
 Enter the name of the new dialog or change the name of the opened dialog
Titlebar Text
 Text of the dialog titlebar
Position 1
 X position of the dialog
Position 2
 Y postition of the dialog
Size 1
 W size of the dialog
Size 2
 H size of the dialog
Use DBU
 If it is on, the dialog has DBU size (Only for dialog size and position, not for controls)
Icon
 Browse icon for the dialog (In the editbox below is showed the selected icon's location)
