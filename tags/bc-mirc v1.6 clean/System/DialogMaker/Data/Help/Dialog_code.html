Description

 Copies the selected text

 Pastes the clipboard text into the end of the dialog code

-
Go To Line
 Opens GoTo Line dialog. From it you can jump to any line from the dialog code. The GoTo Line dialog has a line history too.
Explorer
 Shows/hides the Explorer Bar (The bar which appears in the left side of the Code Editor)
-
  
Patterns
 Shows the menu with patterns. You can create and edit patterns from the Options.

 Checks your code for bracket errors.
