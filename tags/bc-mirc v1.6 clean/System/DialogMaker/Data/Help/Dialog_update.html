Control
Description
Progress
 Shows the current downloading process (In percents)
Time
 When the downloading is completed here appears total time taken for the downloading
Update Size
 Shows size of the downloading update
Downloaded
Shows how many bytes from the current update are already downloaded
Download
Downloads selected update from the list above (You can download any update by double-clicking on it)
Stop
 Stops the downloading process
Refresh
 Refreshes the list and shows the newest information
Current
Shows the current Dialog Maker version
