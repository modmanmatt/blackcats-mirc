What is Dialog Maker

Dialog Maker is a tool for creating and editing dialogs. You can create dialog, set it's position and size, add controls and manipulate them. Now you can easily make and modify dialogs with very small knowledge in mIRC dialog scripting. This tool is for everyone, who wants to create his dialogs fast with great and professional design. Dialog Maker includes Code Editor and Dialog Grabber™. It supports modules,skins and languages too. You can save and load your projects and open Dialog Studio(R) files.
