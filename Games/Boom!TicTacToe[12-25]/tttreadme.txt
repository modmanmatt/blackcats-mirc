To load the game run mIRC then type /load -rs "<filepath>\Boom! TicTacToe.mrc".
To unload it - type /unload -rs "<filepath>\Boom! TicTacToe.mrc", or use the menu from menubar.
Afterwards you'll recieve a text how to run the game.

The main menu in the game is at the upper left corner of the window.

X button closes the game.
_ button minimizes the game.
O button sets/unsets ontop setting of the window.
S button turns the sounds on/off.



For more Boom!®Games check www.mircscripts.org
