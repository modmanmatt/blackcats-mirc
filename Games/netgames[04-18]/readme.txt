load the netgames.mrc to to your remote with:
/load -rs path\to\netgames.mrc 

To play a game:
1. You must be connected to the net
2. double click it in the list.

To Add a game to the list:
1. Goto "Options"
2. Enter the Games name 
3. Enter the url to the game
4. Click the "Add" button

To Edit a game:
1. Select a game in the list to edit
2. Goto "Options"
3. Edit the information you wish
4. Click the "Apply" button

To Delete a game:
1. Select the game in the list to delete
2. Goto "Options"
3. Click "Delete a Game"
