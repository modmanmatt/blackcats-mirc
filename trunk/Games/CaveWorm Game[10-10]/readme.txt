CaveWorm Game made by Illusionz

Put the unzipped file in your mirc directory then load into scripts using
/load -rs worm.mrc

Type /worm or select CaveWorm from the commands menu 
to start the game and follow the instructions.


Controls:

Up and Down move the worm
