Description
Control
Common
Text
 Sets the text of the selected control
Size (1,2,3,4)
 Sets the postition and size of the selected control (First - X, Second - Y, Third - W and Forth - H)
Disable
 Disables the selected control (The control can't be used)
Hide
 Hides the selected control (The control is hidden and you can't show it and hide it again with commands: /did -v [dialog] [control id] to show and /did -h [dialog] [control id] to hide
Result
 The setted function of the selected control will be done after closing the whole dialog
Group
 The control is a part of a group - that means if it's check,radio box or something else when you click on it to set some option you have first to click on the button with OK option to save it.
Type
 Some kind of control have different additional options - Type. These controls are: Buttons, Combo boxes, Check buttons, Radio buttons and Scrollbars
Align
 The Editboxes, Texts, Check buttons, Radio buttons and Icons have an align of the text or the image. The aligns are: Left, Right and Center
Tab
 Set the tab there to show the selected control
Controls
 Select control from the list, which you will edit
Copy
 Copies and after that pastes the selected control from the list above
Delete
 Deletes the selected control from the list above
Save
 Saves the edited options and settings

Password Entry
The letters of the text in the editbox are as stars (*). Usuallyused for passwords
Allow Return
If the editbox is bigger than one line and you want to go to the next line with pressing 'Enter' use this option
Read-Only
 This option is like 'Disable' but if you use this option the text from the editbox can be copied and the text color is normal (If the editbox is disabled the color of the text is gray)
Auto H Scroll
 Auto horizontal scrolling (If the text is longer than the editbox length use this option to increase text length)
Auto V Scroll
 Same as the option above but now the auto scrolling is vertical
Horizontal Scroll
Shows horizontal scrollbar in the editbox
Vertical Scroll
 Shows vertical scrollbar in the editbox
Limit
 Limits the length of the symbols entered in the editbox
